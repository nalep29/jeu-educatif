Petit jeu éducatif pour jeunes enfants, réalisé en dart.

Projet réalisé en décembre 2012 pour le cours SIO-6014 : Applications web des SIO
Auteur : Nadège Le Pironnec
Disponible sous git-hub : https://github.com/nalep29/jeu-educatif.git

L'enfant doit d'abord cliquer sur l'image correspondant au mot affiché en haut de l'écran. 
Quand il a choisi la bonne image, il doit ensuite réécrire le mot en dirigeant un petit vaisseau (poisson) sur les bonnes lettres.

(Bien entendu, le jeu étant destiné à de jeunes enfants ne sachant pas bien lire, un adulte pourra expliquer le jeu à l'enfant la première fois.)

Le jeu n'est pas terminé, on pourrait y ajouter idéalement les fonctionnalités suivantes :
- un bouton 'rejouer' quand la partie est finie
- le score (?)
- de la musique (et/ou des sons d'animaux) quand l'enfant gagne ou perd
- des niveaux de difficulté : mots plus ou moins longs, vitesse du vaisseau plus ou moins rapide, etc
- rajouter d'autres catégories (pour avoir autre chose que des ANIMAUX) : NOURRITURE / VÊTEMENTS / VÉHICULES, etc
